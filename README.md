## Report a bug

Found a problem with Sol's Relic System? [Open a structured bug report](https://github.com/markmochi/sols-relic-system-wiki/issues/new?template=bug_report.yml) or read the [complete reporting guide](https://markmochi.github.io/sols-relic-system-wiki/reference/reporting-issues/).

Before submitting:

1. Search [existing issues](https://github.com/markmochi/sols-relic-system-wiki/issues) for the same problem.
2. Check the [FAQ and troubleshooting guide](https://markmochi.github.io/sols-relic-system-wiki/reference/troubleshooting/).
3. Reproduce the problem once more after restarting the game or server.
4. Use a clear title describing what failed and where it happened.

Every report should include:

- the exact Minecraft version;
- Forge or NeoForge and its complete version;
- the exact Sol's Relic System and Sol's Item Rarity versions;
- singleplayer, LAN, or dedicated-server environment;
- numbered reproduction steps;
- the expected result and actual result;
- `logs/latest.log` or the matching crash report;
- other installed mods, modpack version, datapacks, and relevant configuration changes;
- full-tooltip and Aster Table screenshots when relic stats, leveling, ascension, or rerolls are involved.

Remove access tokens, server addresses, player IP addresses, personal paths, and other private information before uploading logs.
