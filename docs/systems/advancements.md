# Resonance advancements

The v1.55 resource code adds an advancement chain driven by inventory possession.

| Advancement | Frame | Trigger |
|---|---|---|
| First Spark of Resonance | Task | Obtain Resonance Core I |
| Echoes Gather | Task | Obtain Resonance Core II |
| Harmonic Current | Goal | Obtain Resonance Core III |
| Gilded Resonance | Goal | Obtain Resonance Core IV |
| Mythic Convergence | Challenge | Obtain Resonance Core V |
| Supreme Resonance | Challenge | Obtain Resonance Core VI |
| Fate, Rewritten | Challenge | Obtain Dust of Enlightenment |
| Sixfold Sovereign | Challenge | Hold all six Resonance Core levels at once |

All are visible, show a toast, and announce in chat. The chain tracks obtaining materials, not successfully performing the associated ascension or reroll.
